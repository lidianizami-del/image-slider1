# Простой слайдер изображений

Интерактивный слайдер изображений, созданный с использованием HTML, CSS и JavaScript.

## Функциональность

- Навигация с помощью кнопок "вперед" и "назад"
- Зацикливание слайдера
- Отображение текущего номера изображения
- Навигация с помощью точек
- Автопрокрутка с регулировкой скорости
- Управление с клавиатуры (стрелки влево/вправо, пробел)
- Адаптивный дизайн для мобильных устройств

## Использование

1. Склонируйте репозиторий
2. Откройте файл `index.html` в браузере
3. Используйте кнопки или клавиши для навигации

## Технологии

- HTML5
- CSS3 (Flexbox, анимации, медиазапросы)
- Vanilla JavaScript
- Font Awesome для иконок
- Unsplash для изображений
